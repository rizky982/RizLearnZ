Catatan perjalanan belajar dunia IT dan jaringan!

Dari Nol Sampai Paham

Saya memulai belajar komputer dan jaringan dari benar-benar nol. Awalnya banyak error, bingung, bahkan sempat menyerah. Tapi dengan semangat dan rasa ingin tahu, perlahan saya mulai memahami dasar-dasar seperti instalasi sistem operasi, membuat jaringan lokal, dan konfigurasi server.

Proyek & Karya

Setelah cukup paham teori, saya mulai membuat proyek kecil seperti membuat website HTML/CSS, konfigurasi FTP server di Debian, dan mencoba membuat web dinamis dengan PHP + MySQL. Dari sini saya belajar banyak hal, termasuk cara menyelesaikan error yang muncul.

Lomba & Sertifikasi

Saya juga sempat ikut lomba jaringan di tingkat kota dan sedang dalam proses belajar untuk ikut sertifikasi jaringan dari Cisco. Semua ini jadi bagian penting dari perjalanan belajar saya sebagai anak TKJ.
