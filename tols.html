Berikut ini adalah daftar tools dan referensi penting yang bisa membantumu dalam proses belajar di jurusan TKJ:

1. Visual Studio Code

Editor kode ringan namun powerful untuk menulis HTML, CSS, JavaScript, dan bahasa pemrograman lainnya. Download di code.visualstudio.com.

2. draw.io / diagrams.net

Tools gratis untuk membuat diagram jaringan dan flowchart. Sangat membantu untuk perencanaan topologi jaringan.

3. GitHub

Tempat menyimpan, membagikan, dan mengelola proyek coding secara online menggunakan Git. Bisa juga digunakan untuk portofolio.

4. Packet Tracer

Simulasi jaringan dari Cisco. Cocok untuk latihan konfigurasi jaringan virtual sebelum praktik langsung dengan perangkat nyata.

5. Termux (Android)

Aplikasi terminal untuk menjalankan perintah Linux langsung dari HP Android. Cocok untuk belajar dasar CLI.

6. DokuWiki / Markdown

Gunakan format Markdown atau Wiki untuk mencatat materi. Ini memudahkan dokumentasi materi secara terstruktur.

7. Free Hosting (000webhost, InfinityFree)

Tempat belajar upload website secara gratis. Cocok untuk latihan hosting HTML atau CMS seperti WordPress.

8. SSH Tools (PuTTY / Termius)

Tools untuk akses server Linux jarak jauh menggunakan SSH. Sangat penting saat belajar administrasi server.

9. GNS3 atau EVE-NG

Tools virtualisasi jaringan profesional. Cocok untuk level lanjut dalam simulasi jaringan menggunakan router dan switch sungguhan (virtual).

10. YouTube Edu Tech

Channel seperti Kelas Terbuka, WPU, atau TechWithTim bisa bantu kamu belajar coding dan networking dengan mudah.

11. ChatGPT / AI Assistant

Gunakan AI seperti ChatGPT untuk tanya jawab seputar error, belajar konsep jaringan, atau buat skrip sederhana. Gunakan dengan bijak.

12. Buku TKJ / PDF Materi

Jangan lupakan buku cetak dan modul dari guru. Kombinasi belajar dari buku dan praktik langsung akan sangat membantu pemahamanmu.

Tips: Selalu update tools yang kamu gunakan dan biasakan dokumentasi setiap belajar hal baru ya!
