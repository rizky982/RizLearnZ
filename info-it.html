1. Apa Itu Cloud Computing?

Cloud Computing adalah teknologi yang memungkinkan kamu menyimpan, mengakses, dan mengelola data atau aplikasi lewat internet, tanpa harus memiliki server fisik sendiri. Contohnya seperti Google Drive, Dropbox, atau layanan AWS (Amazon Web Services). Keuntungannya adalah hemat biaya, fleksibel, dan bisa diakses dari mana saja.
