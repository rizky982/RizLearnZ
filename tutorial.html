1. Cara Install VirtualBox

VirtualBox adalah software untuk menjalankan sistem operasi virtual. Unduh dari situs resminya, lalu install seperti biasa. Setelah itu, kamu bisa mulai membuat mesin virtual untuk belajar jaringan tanpa harus menggunakan perangkat fisik.

2. Cara Install Debian di VirtualBox

Setelah VirtualBox terpasang, buat mesin virtual baru, pilih sistem operasi Debian. Gunakan file ISO Debian saat instalasi, atur partisi, dan ikuti panduan hingga selesai.

3. Konfigurasi IP Address

Buka terminal di Debian, lalu gunakan perintah nano /etc/network/interfaces untuk mengatur IP secara manual. Misalnya: address 192.168.1.2 dan netmask 255.255.255.0.

4. Instalasi FTP Server

Gunakan perintah apt install vsftpd. Setelah itu, konfigurasi file /etc/vsftpd.conf agar client bisa mengakses file lewat FileZilla.

5. Instalasi Web Server (Apache)

Pasang dengan apt install apache2. Setelah selesai, buka browser dan akses IP server. Jika muncul halaman default Apache, berarti server berhasil dijalankan.

6. Upload Website ke Web Server

Letakkan file HTML ke folder /var/www/html. Gunakan perintah cp atau upload via FTP. Pastikan permission file sudah sesuai agar bisa diakses dari browser.

7. Konfigurasi DNS Server

Install paket bind9, lalu konfigurasi file zone dan named.conf.local. DNS memungkinkan kamu mengakses server menggunakan nama domain lokal seperti www.rizlearnz.com.

8. Instalasi DHCP Server

Gunakan apt install isc-dhcp-server. Atur file konfigurasi /etc/dhcp/dhcpd.conf dengan rentang IP yang ingin diberikan secara otomatis ke client.

9. Konfigurasi Mikrotik Dasar

Hubungkan Mikrotik ke laptop via Winbox. Atur IP, NAT, dan DHCP Server dari menu IP > Addresses, DHCP Server, dan NAT. Mikrotik membantu mengatur jaringan secara profesional.

10. Konfigurasi Jaringan LAN

Sambungkan beberapa PC menggunakan switch dan atur IP masing-masing. Pastikan semua berada di satu jaringan agar bisa saling terhubung. Tes dengan perintah ping.

11. Uji Koneksi Jaringan

Lakukan tes koneksi dengan ping. Contoh: ping 192.168.1.1. Jika balasan muncul, koneksi berhasil. Jika tidak, cek kabel, IP, dan pengaturan firewall.
